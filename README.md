# 📝 Survey Form - Hábitos Saudáveis

Este é um projeto de formulário interativo desenvolvido com **React.js**. O objetivo é coletar dados sobre hábitos de saúde dos usuários, apresentando uma interface moderna e responsiva.

## 🎨 Layout

O formulário possui um design focado no bem-estar, utilizando tons de verde oliva e um fundo escuro para reduzir o cansaço visual.
